树莓派USB启动

发表于 2016-07-04   |   分类于 Server

概述

　　一般来说很多人玩树莓派系统都是放在TF卡上的，但是TF相比较U盘的差距还是挺大的，例如速度，寿命，容量等，接下来我们将使用U盘来替代TF卡成为我们树莓派的主存储

准备工具

  1. Mac一台（其他系统差不多
  2. kali镜像（其他的镜像差不多，不一定非要kali）
  3. TF卡一张（在满足树莓派要求的TF下买最便宜的，我的是4GB的，只做引导USB启动而已）
  4. U盘一个 (64GB）

操作步骤

格式化TF卡和U盘

打开Disk Utility进行格式化如图1、2

0x02 查看设备描述

打开终端 diskutil list查看设备描述如图2这里我的TF卡是/dev/disk3 u盘是/dev/disk4 如图3

0x03 unmout u盘设备

diskutil unmountdisk /dev/disk4 如图4

0x04 烧写镜像

sudo dd if=/pathtoyour/kali-2.1.2-rpi2.img of=/dev/disk4 bs=1m 如图5

0x05 修改启动配置文件

将烧写好的的U盘打开并将所见所有文件复制到tf卡中，并修改tf卡中的cmdline.txt文件将root=/dev/mmcblk0p2 修改为/dev/sda2 保存如图6、7

0x06 重分配空间

把U盘和TF卡从电脑上拔出插入树莓派并启动，df -h 发现我们的U盘并没达到64G，而只有可怜的6.7G，重新分配空间，如图8，9，10

重新启动 reboot

Enjoy!
